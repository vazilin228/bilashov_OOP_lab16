# Лабораторна робота №16 — Геометричні фігури

## Завдання №1
Система класів для опису плоских геометричних фігур: **Коло**, **Квадрат**, **Прямокутник**.

---

## Структура проекту

```
GeometryApp/
├── GeometryApp.csproj
├── Program.cs
├── Form1.cs              ← Windows Forms головна форма
└── Models/
    ├── Shape.cs          ← Абстрактний базовий клас
    ├── Circle.cs         ← Клас «Коло»
    ├── MySquare.cs       ← Клас «Квадрат»
    └── MyRectangle.cs    ← Клас «Прямокутник»
```

---

## Вимоги лабораторної (реалізовано)

| Вимога                    | Де реалізовано                                                             |
|---------------------------|----------------------------------------------------------------------------|
| **3 конструктори**        | Кожен клас має конструктор без параметрів, з розміром, з позицією+кольором |
| **Властивості**           | X, Y, Angle, Color + Radius / Side / Width, Height                        |
| **Статичні елементи**     | `Shape.TotalCount` — лічильник всіх створених фігур                       |
| **Перевантаження методів**| `Move(int,int)` / `Move(double,double)` / `Move(double,double,bool)`      |
|                           | `Resize(double)` / `Resize(double,bool)` / `Resize(double,double)` (Rect) |
| **Переміщення**           | `Shape.Move(dx, dy)`                                                       |
| **Зміна розмірів**        | `Shape.Resize(factor)` — масштабування; Rectangle підтримує незалежне     |
| **Поворот**               | `Shape.Rotate(degrees)` — накопичувальний кут                             |

## Опис інтерфейсу

**Ліва панель:**
- Вибір типу фігури (коло / квадрат / прямокутник)
- Координати центру X, Y
- Розміри (радіус / сторона / ширина+висота)
- Вибір кольору
- Кнопка «Додати фігуру»

**Операції (ліва панель):**
- Переміщення на dX, dY
- Поворот на заданий кут
- Масштабування у відсотках (100% = без змін)
- Видалення обраної фігури / очищення всіх

**Полотно (права частина):**
- Відображення фігур з сіткою
- Золотий ореол навколо виділеної фігури
- Нумерація фігур

**Список фігур (внизу):**
- Показує тип, розміри, позицію, кут, площу, периметр
